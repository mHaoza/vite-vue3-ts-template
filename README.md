## Vite + Vue3 + Ts template
自用Vite项目模板  
Vite + Vue3 + Ts + VueRouter + Pinia  
eslint + prettier  
husky + lint-staged + commitizen  

### init project
1. 拉取模板
  ``` shell
    degit forminiice/vite-vue3-ts-template your-project-name
  ```
2. 初始化git
  ``` shell
    git init
  ```
3. 安装依赖
  ``` pnpm
    pnpm i
  ```
